1. Download app
2. Download Firefox
3. Open firefox:

   - Create profile and copy path profile
   - Login Tiktok, Facebook (change profile), Youtube => if config.json active is true.
4. Edit file config.json (edit link path profile firefox)
5. Start file runs.exe
